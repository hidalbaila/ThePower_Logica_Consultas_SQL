# ThePower_Lógica_Consultas_SQL
En este proyecto tengo que aplicar los conocimientos aprendidos en el módulo de SQL.


## Requisitos del proyecto
Las consultas pueden hacerse de distintas maneras para obtener el mismo resultado. A lo largo del proyecto debes demostrar los siguientes conocimientos:
  ● Manejo de la herramienta DBeaver.
  ● Manejo de las consultas con una sola tabla de la BBDD.
  ● Manejo de las relaciones entre tablas.
  ● Manejo de las subconsultas.
  ● Vistas.
  ● Estructura de datos temporales.
  ● Buenas prácticas.
